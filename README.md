
# Group Chatting app :tada:

A mutli-threaded whatsapp alike offline desktop chatting application .

## Technology & Tools used :desktop_computer:

* Socket programming in Java 
* Multi-threading in Java
* JAVA SWT(Swing+AWT)
* Netbeans IDE
